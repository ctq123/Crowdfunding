# crowdfunding
一个完整的应用，Ganache + Truffle开发智能合约，用户界面DApp，实现众筹功能

## 智能合约

#### 前置条件
若还没有安装truffle和ganache-cli的需要提前安装

```bash
npm install -g truffle@6.14.13
npm install -g ganache-cli@6.14.13
```
> node版本说明
> node@16.14.0

#### 编译
```
truffle compile
```

#### 启动
新建一个窗口，启动服务
```
ganache-cli
```

#### 部署
```
truffle migrate
```


## 安装
```
yarn
```

## 运行
```
yarn serve
```
